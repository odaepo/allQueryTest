Three query

This is an initial interface for buildiong a query (using Doctrine ORM).
Suppose you has Ta... Tz Tables.
You can ask (BUILD) a query starting only fron T1, T2, T3, t9
T1 is connected (one to many or one to one) to tables T4, T5, T6
T2 is connected to tables T7, T8
T3 is connected with no tables.
T4 to T9,T10
T5 to T11
T6 to none
T7, T8, T10,T11, are connected to no table.
T11 is connected with T1 (creating a loop!)
